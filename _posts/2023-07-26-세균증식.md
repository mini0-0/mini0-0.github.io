## 3. 다른 사람 풀이
![image](https://github.com/mini0-0/mini0-0.github.io/assets/63296983/11687e6c-4362-4216-81a8-cf17ec54cdd2)

**메모**

- 다른 사람 풀이를 보면 shift 연산자 사용함(사용 잘안해서 생각하지 못함)
- 왼쪽 시프트 연산자 ( < < )
    - 2를 곱한 것과 같은 효과
    - n << m
        - n * 2의 m승
- 오른쪽 시프트 연산자 ( > > )
    - 2로 나눈 것과 같은 효과
    - n >> m
        - n / 2의 m승
