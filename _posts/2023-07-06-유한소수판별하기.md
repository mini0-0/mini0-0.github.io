## 2. 내 풀이

```python
from math import gcd

def solution(a, b):
    b//=gcd(a,b)
    while b%2==0:
        b//=2
    while b%5==0:
        b//=5
        
    return 1 if b==1 else 2
```

**풀이**

1. 힌트에 나온 **“분자와 분모의 최대공약수로 약분하면 기약분수를 만들 수 있습니다.”** 활용하기 위해 gcd 불러옴
2. 기약분수 일때 분모의 소인수 값이 [2,5]만 존재해야함 

→ 즉, 2 or 5로만 나눠 떨어져야함 

1. b==1이면 기약분수&유한소수임

## 3. 다른사람 풀이

```python
from math import gcd

def solution(a, b):
    b = b / gcd(a, b)
    for i in [2, 5]:
        while not b % i:
            b //= i

    return 1 if b == 1 else 2
```

**메모**

while not 구문 - while not 구문이 True일 동안 실행한다.
