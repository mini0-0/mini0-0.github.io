**메모**

 1.  zfill(width) 함수 사용

```python
"2".zfill(3)`
"002"`

`"50000".zfill(5)`
"50000"`

"123".zfill(5)`
"00123"`
```

2. replace 함수 활용

```python
>>> a = 'hello world'
>>> a.replace('hello','hi')
hi world
```

## 3. 다른 사람 풀이
![image](https://github.com/mini0-0/mini0-0.github.io/assets/63296983/1f30a0a8-d3b7-4a1b-a97c-fac122073eb4)

1. **rjust(width, [fillchar]) 함수** 사용

```python
"2".rjust(3, "0")
#"002"`

"50000".rjust(5, "0")
#"50000"

"123".rjust(5, "0")
#"00123"

"123".rjust(5, "a")
#"aa123"
```
